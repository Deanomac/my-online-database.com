Collections

In My Online Database, your tables are called "Collections." You can create multiple collections to organize your data based on different categories or purposes.

Storing Images

Need to store images and access them easily? We've got you covered! Follow these simple steps:

  1. Sign up for a free account on Imgur.
  2. Upload your image to Imgur.
  3. Right-click on the uploaded image and select "Open image in new tab."
  4. Copy the URL from the address bar of the new tab.
  5. Paste the copied URL wherever you want to use the image in My Online Database.

Form View

Switch to the Form View to see a more detailed view of each record. The Form View allows for easier editing of individual records and provides navigation between records.

Search and Filtering

Use the search bar to find specific records. The search function looks through all fields in the table. You can also filter records by specific columns using the filter inputs in the table headers.

Sorting Data

Sort the data by clicking on the column headers. Clicking once sorts ascending, clicking again sorts descending. The current sort column and direction are indicated by an arrow icon.

Editing Data

Edit data directly in the table view by double-clicking on a cell. Changes are saved automatically when you click out of the cell or press Enter. You can also edit data in the Form View for a more focused editing experience.

Keyboard Navigation

Navigate between cells in the table using the arrow keys. Pressing Enter on a cell enters edit mode, and pressing Esc cancels the edit. Use Tab and Shift+Tab to move between fields in the Form View.

Undo/Redo

Undo your last action using Ctrl+Z (or Cmd+Z on Mac). Redo an undone action using Ctrl+Y (or Cmd+Y on Mac). The undo/redo functionality applies to most data editing actions.

Managing Columns

My Online Database provides easy column management options. Simply right-click on a column header to access the following options:

  • Rename Column: Change the name of the column to better suit your needs.
  • Hide Column: Hide the column from view if you don't need it displayed.

Deletion Safety

We understand that accidental deletions can be frustrating. That's why we've implemented safety measures to prevent unintended data loss:

  • Delete Table: The "Delete Table" button is disabled by default. You need to toggle the settings to enable it, ensuring you don't accidentally delete an entire table.
  • Delete Row: Row deletion is also disabled by default. You need to toggle the settings to make the row delete option appear.
  • Delete Column: To delete a column, you first need to toggle the settings for column visibility. This extra step helps prevent accidental column deletions.

Toggling Settings

To access the deletion options and other advanced features, you need to toggle the settings. Here's how:

  1. Click on the "Settings" button in the top navigation bar.
  2. In the settings panel, look for the "Enable Deletion" option.
  3. Toggle the switch to enable or disable deletion options.
  4. Close the settings panel, and the corresponding deletion buttons will appear based on your settings.

Importing and Exporting Data

My Online Database supports importing and exporting data in various formats, making it easy to migrate your data from other sources or back up your collections.

Importing Data

  1. Click on the "Import" button in the top navigation bar.
  2. Choose the format of your data file (CSV, TSV, Google Sheets, etc.).
  3. Select the file from your computer or provide the URL if importing from a web source.
  4. Map the columns in your file to the corresponding fields in your collection.
  5. Click "Import" to start the import process.

Exporting Data

  1. Open the collection you want to export.
  2. Click on the "Export" button in the top navigation bar.
  3. Choose the desired export format (CSV, TSV, JSON, etc.).
  4. Specify any additional export options, such as field delimiter or including headers.
  5. Click "Export" to download the exported file.

Customizing Themes

My Online Database allows you to personalize your workspace with different themes. To change your theme:

  1. Click on the "Settings" button in the top navigation bar.
  2. In the settings panel, navigate to the "Appearance" section.
  3. Select your desired theme from the available options.
  4. The new theme will be applied automatically.

Keyboard Shortcuts

Boost your productivity with these handy keyboard shortcuts:

  • Ctrl + F or Cmd + F: Search within a collection.
  • Ctrl + N or Cmd + N: Create a new collection.
  • Ctrl + S or Cmd + S: Save changes to a record.
  • Ctrl + Z or Cmd + Z: Undo the last action.
